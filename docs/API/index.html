Confslice

Confslice - A simple configuration file parser

Confslice is a C++ library for parsing structured configuration files and generating configuration schemes. It defines a simple and compact configuration file format which consists of entities and keys. Entities represent independent components such as subsystems, while keys represent characteristics and can be associated with an entity or the global configuration.

Building and installing Confslice

To build confslice just type: make

To build a development version of Confslice with support for debugging symbols type: make dev

To build only the tests type: make tests

To install the confslice library, run the following as root: make install

Removing Confslice

To remove confslice just type as root: make remove

Testing Confslice

We have included several unit tests which check whether the confslice library works correctly. Check the tests folder for details.
